Statistical Hypothesis Testing Problems

📊 문제 1: 일표본 t검정 (One-sample t-test)

어떤 제약회사에서 생산하는 비타민 C 정제의 비타민 C 함량이 평균 500mg이라고 주장하고 있다. 이를 검증하기 위해 20개의 표본을 무작위로 선택하여 비타민 C 함량을 측정했다.

α = 0.05 수준에서 제약회사의 주장이 맞는지 검정하시오. (양측검정)

가설 설정

  • H₀: μ = 500 (제약회사의 주장이 맞다)
  • H₁: μ ≠ 500 (제약회사의 주장이 틀리다)

검정 조건

  • 유의수준: α = 0.05
  • 검정방법: 양측검정
  • 표본크기: n = 20

📊 문제 2: 대응표본 t검정 (Paired t-test)

새로운 다이어트 프로그램의 효과를 검증하기 위해 15명의 참가자를 대상으로 프로그램 시작 전과 후의 체중을 측정했다.

α = 0.05 수준에서 다이어트 프로그램이 체중 감소에 효과가 있는지 검정하시오. (단측검정: 체중이 감소했는가?)

가설 설정

  • H₀: μd ≤ 0 (다이어트 프로그램의 효과가 없다)
  • H₁: μd > 0 (다이어트 프로그램이 체중 감소에 효과가 있다)

여기서 d = 다이어트 전 체중 - 다이어트 후 체중

검정 조건

  • 유의수준: α = 0.05
  • 검정방법: 우측 단측검정
  • 표본크기: n = 15

📊 문제 3: 독립표본 t검정 (Independent samples t-test)

두 개의 서로 다른 교육 방법(A방법, B방법)의 효과를 비교하기 위해 A방법으로 교육받은 학생 12명과 B방법으로 교육받은 학생 15명의 시험 점수를 측정했다.

α = 0.05 수준에서 두 교육 방법 간에 평균 점수의 차이가 있는지 검정하시오. (양측검정)

가설 설정

  • H₀: μA = μB (두 교육 방법의 평균 점수가 같다)
  • H₁: μA ≠ μB (두 교육 방법의 평균 점수가 다르다)

검정 조건

  • 유의수준: α = 0.05
  • 검정방법: 양측검정
  • 가정: 등분산성
  • 표본크기: nA = 12, nB = 15

📊 문제 4: 일원배치 분산분석 (One-way ANOVA)

세 가지 서로 다른 비료(A, B, C)가 식물의 성장에 미치는 영향을 조사하기 위해 각 비료를 사용한 식물의 키를 측정했다.

  • A비료: 10개체
  • B비료: 12개체
  • C비료: 11개체

α = 0.05 수준에서 세 비료 간에 평균 식물 키의 차이가 있는지 검정하시오.

가설 설정

  • H₀: μA = μB = μC (세 비료의 평균 효과가 같다)
  • H₁: 적어도 하나의 평균이 다르다

검정 조건

  • 유의수준: α = 0.05
  • 검정방법: F검정
  • 표본크기: nA = 10, nB = 12, nC = 11

📋 참고사항

기본 가정

  1. 모든 검정에서 정규성과 독립성을 가정합니다.
  2. t검정에서는 t분포를, ANOVA에서는 F분포를 사용합니다.
  3. 각 검정의 검정통계량, p-value, 임계값을 계산하세요.
  4. 유의수준과 p-value를 비교하여 결론을 내리세요.
  5. 실제 상황에서의 의미를 해석하세요.

✅ 계산해야 할 항목들

  • ☐ 표본 통계량 (평균, 표준편차)
  • ☐ 검정통계량 (t값 또는 F값)
  • ☐ 자유도
  • ☐ p-value
  • ☐ 임계값
  • ☐ 결론 (귀무가설 기각/채택)
  • ☐ 실제적 해석

📐 검정 공식 모음

1. 일표본 t검정

t = (x̄ - μ₀) / (s / √n)
자유도: df = n - 1

2. 대응표본 t검정

t = d̄ / (sd / √n)
여기서 d̄ = 차이값들의 평균, sd = 차이값들의 표준편차
자유도: df = n - 1

3. 독립표본 t검정 (등분산 가정)

t = (x̄₁ - x̄₂) / (sp√(1/n₁ + 1/n₂))
합동표준편차: sp = √[((n₁-1)s₁² + (n₂-1)s₂²) / (n₁+n₂-2)]
자유도: df = n₁ + n₂ - 2

4. 일원배치 분산분석

F = MSB / MSW
MSB = SSB / (k-1)
MSW = SSW / (N-k)
SSB = Σnᵢ(x̄ᵢ - x̄)²
SSW = ΣΣ(xᵢⱼ - x̄ᵢ)²
자유도: df₁ = k-1, df₂ = N-k
